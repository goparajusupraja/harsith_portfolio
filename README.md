# 💼 Personal Portfolio Website

Welcome to my Personal Portfolio Website! This project showcases my skills, education, projects, and career objectives as a Computer Science Engineering student. It is designed with a modern, responsive, and user-friendly interface to provide visitors with an overview of my professional profile.

## ✨ Features

* Modern and responsive design
* Interactive hero section with typing animation
* About Me section
* Educational qualifications timeline
* Technical skills with animated progress bars
* Projects showcase
* Strengths and hobbies section
* Contact information
* Smooth scrolling navigation
* Scroll reveal animations
* Dark-themed professional UI
* Mobile-friendly layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 Objective

The purpose of this portfolio is to present my academic background, technical skills, and projects while demonstrating my front-end web development abilities. It also serves as my professional portfolio for internships, placements, and software engineering opportunities.
